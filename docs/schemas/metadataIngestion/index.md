
# Metadata Ingestion

Schemas for metadata extraction and ingestion workflows.

## Overview

Ingestion workflows extract metadata from source systems and load into OpenMetadata.

## Workflow Schema
**Schema**: `schemas/metadataIngestion/workflow.json`

## Components

### Source
Configure metadata source.

### Sink  
Configure metadata destination.

### Processor
Transform metadata.
