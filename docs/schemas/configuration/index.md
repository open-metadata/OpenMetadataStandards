
# Configuration Schemas

System configuration schemas for OpenMetadata deployment.

## Configuration Categories

### Authentication
- LDAP, SAML, OAuth, OIDC configurations

### Authorization  
- RBAC and policy configurations

### Infrastructure
- Database, Elasticsearch, Kafka settings

### Pipeline Services
- Airflow, Prefect integration

### Event Handlers
- Webhook and notification configurations
