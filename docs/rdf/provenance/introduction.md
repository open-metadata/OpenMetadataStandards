
# Provenance Ontology Introduction

OpenMetadata provenance tracking using W3C PROV-O.

**File**: `rdf/ontology/openmetadata-prov.ttl`

## Overview

Tracks data lineage and provenance using PROV-O standard.

## PROV-O Concepts

### Entity
Data assets (tables, files).

### Activity
Processes (pipelines, queries).

### Agent
Users, teams, systems.
