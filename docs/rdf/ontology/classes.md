
# Ontology Classes

Class definitions in the OpenMetadata ontology.

## Core Classes

### om:Entity
Base class for all entities.

### om:DataAsset
Data-bearing entities.

### om:Table
Database tables.

### om:Dashboard
BI dashboards.

## Class Hierarchy

All classes use rdfs:subClassOf for inheritance.
