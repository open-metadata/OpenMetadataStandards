
# Metadata Standards

Industry metadata standards and compliance.

## Overview

OpenMetadata Standards aligns with and supports multiple industry standards for metadata management.

## Supported Standards

### JSON Schema
- Draft-07 (current)
- Draft 2020-12 (planned migration)

### W3C Standards
- RDF 1.1
- OWL 2
- SHACL
- JSON-LD 1.1
- PROV-O

### OpenAPI
API specifications using OpenAPI 3.0.

### Industry Standards
- DCAT (Data Catalog Vocabulary)
- Dublin Core
- Schema.org

## Compliance

OpenMetadata ensures compliance with:
- Data privacy regulations (GDPR, CCPA)
- Industry regulations (HIPAA, SOC 2)
- Enterprise standards
