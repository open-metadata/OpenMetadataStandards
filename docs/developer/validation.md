
# Validation

Validating schemas and data.

## JSON Schema Validation

Use standard validators:
- Python: jsonschema
- JavaScript: ajv
- Java: everit-json-schema

## RDF Validation

Use SHACL validators:
- Python: pyshacl
- Java: Apache Jena

## Continuous Validation

GitHub Actions run validation on every commit.
