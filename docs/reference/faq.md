
# Frequently Asked Questions

Common questions about OpenMetadata Standards.

## General

**Q: What is OpenMetadata Standards?**  
A: Comprehensive schemas and standards for metadata management.

**Q: How do I validate against schemas?**  
A: Use JSON Schema validators with the schema files.

## Schemas

**Q: How many schemas are included?**  
A: 700+ JSON schemas plus RDF ontologies.

**Q: Are schemas versioned?**  
A: Yes, using semantic versioning.
